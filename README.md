일상 생활의 새로운 것들? 문화의 상징인 발명품!
알고 보면 일상은 과거에 비해 독특한 것들로 가득하다.

B.C 1만 6천년 전 부터 현재까지 라이프 스타일에
왕창 변화를 준 발명품들을 소개하는 웹사이트.

발명품의 역사를 확인하고, 발명품에 대한 당신에 생각을 
나누어 보세요.

발명품의 역사를 _**Spanner**_ 와 함께 하세요.
